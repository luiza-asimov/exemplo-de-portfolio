A Asimov Academy é uma escola de programação e Data Science focada no ensino de Python para Dados, com cursos online, suporte dos professores e comunidade para interação entre alunos. Oferecemos cursos de programação em Python, Data Science & Machine Learning, Dashboards e Trading Quantitativo.

Este é um exemplo de portfólio onde você pode inserir suas informações.Vamos deixar ele preenchido com dados da Asimov para você entender como funciona. No nosso repositório do github(LINK) você encontra maiores informações sobre como editá-lo. Não esqueça de conferir nosso post do blog(link) sobre como fazer bom portfólio.
