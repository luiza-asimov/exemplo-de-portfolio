Dashboard "Imóveis de NYC"

Dashboard para análise de vendas em New York.

Projeto que utiliza as bibliotecas Plotly & Dash e Pandas para o desenvolvimento de um dashboard para análise de vendas no mercado imobiliário de Nova Iorque. Para a criação do mapa, foi utilizada a API do Mapbox.

Se você tem interesse em desenvolver esse projeto para adicioná-lo ao seu portfólio, acesse o site da Asimov Academy. Lá você encontra esse projeto e muitos outros!
